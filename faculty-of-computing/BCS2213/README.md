## This is Formal Methods

## Using [Community Z Tools](https://czt.sourceforge.net/)
### New Project
- Open `CZT.exe`
- `File>New>CZT Project`
- Input `Project Name`
- `Finish`

### New Z Specification
- `File>New>Z Specification`
- Input `Name`
- Select `Unicode (encoded as UTF-8)` (Markup)
- `Finish`
